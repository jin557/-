団体紹介 — 理念と使命 —

封祓連は、古より伝わる秘術と現代の知見を融合させ、**人々の日常を脅かす不可解な存在**から守るために結成された、唯一無二の民間祓術師集団です。

我々の活動は、あくまで社会の影に潜み、静かに、そして確実に霊障・呪物・神域の異常を**“封じ、祓う”**ことに徹します。単なる退魔ではなく、原因の究明と根本的な解決を目指し、見えざる闇の均衡を保つことが私たちの使命です。

構成員 — 秘匿された知識と技術 —

封祓連は、日本各地より集いし、選ばれた**祓術師、神職経験者、陰陽道研究者**などで構成されています。

構成員には実名を出さない者も多く、その活動は匿名を原則とします。それぞれの専門分野と深い知識・技術を持ち寄り、見えざる脅威に対し連携して対応にあたります。彼らの存在こそが、封祓連の揺るぎない力です。

宗家直系 祓術師

古来より伝わる封祓の秘術を受け継ぐ、連の中心的存在。強力な霊力を持ち、高難度の案件を統括する。

結界師 / 鑑定士

土地や空間の霊的エネルギーを読み解き、結界の構築・修復を専門とする。呪物の鑑定にも長ける。

怪異研究者 / 資料解析

怪異の発生メカニズムや歴史的背景を学術的に研究。膨大な古文書や現代の情報を解析し、対策を立案する。

護衛 / 現場支援

霊的危険区域での祓術師の安全を確保し、物理的な支援を行う。特殊な訓練を受けた実戦部隊。

個人宅・集合住宅における霊障調査・祓い

業務概要

日常空間で感じる不可解な気配、体調不良、悪夢、金縛りなど、霊的な要因による障がいを調査し、適切な祓術によって解消します。

私たちは、住まいのエネルギーの流れを読み解き、霊障の根本原因を特定。ただ祓うだけでなく、再発防止のためのアドバイスや簡易結界の設置も行います。

実施手順

  1. ご相談・初期ヒアリング（オンラインまたは電話）
  2. 現地調査（霊障の種類、強さ、原因の特定）
  3. 適切な祓術の提案とお見積もり
  4. 祓術の実施（浄化、鎮魂、封印など）
  5. 再発防止のためのアドバイス・簡易結界設置
  6. アフターフォロー

こんな時にご相談ください

  • 引っ越し先で原因不明の体調不良が続く
  • 特定の場所に行くと、寒気や重苦しさを感じる
  • 夜中に不可解な音や声を聞く
  • 家族の間で不和が続く、病気がちである
  • 以前からある土地や建物に不安を感じる

神社・廃寺などの結界再構築

業務概要

古くから聖地とされてきた神社、寺院、またはかつての聖域が、荒廃や人の手によって結界の力が弱まり、悪しきものが侵入する事例が増えています。

封祓連は、土地の歴史と霊脈を読み解き、古伝の結界術を用いて弱体化した結界を修復・再構築します。これにより、地域の守護と平和を回復させます。

実施手順

  1. 土地の霊的調査・歴史的背景の検証
  2. 既存結界の破損状況と霊的汚染度の診断
  3. 結界再構築の計画立案
  4. 儀式と術を用いた結界の修復・強化
  5. 長期的な維持管理に関する助言

こんな時にご相談ください

  • 管理している神社や寺院の敷地内で不審な現象が頻発する
  • 古くからある聖域が荒れ果て、周辺住民への悪影響が懸念される
  • 特定の場所の雰囲気が一変し、人々が近づかなくなった
  • 地域の守り神の力が弱まっていると感じる

呪物の隔離・浄化処理

業務概要

人々の怨念や強い思念が宿り、負のエネルギーを発する「呪物」は、放置すると周囲に甚大な被害を及ぼす可能性があります。

封祓連は、専門の技術と知識をもって呪物を安全に回収し、その力を無力化するための浄化処理や、厳重な隔離・封印を行います。また、呪物が生み出された背景の調査も可能です。

実施手順

  1. 呪物の鑑定・危険度評価
  2. 安全な回収手順の確立と実施
  3. 専用施設での浄化儀式または封印処理
  4. 呪物の保管または適切な処分
  5. 関連する霊的影響の調査と除去

こんな時にご相談ください

  • 所有する物品から不吉な気配を感じる、悪夢を見るようになった
  • 特定の物品の周りで不審な現象が起きる
  • 曰く付きの物品を偶然手に入れてしまい、処分に困っている
  • 人から贈られたものが原因で不幸が続くように感じる

怪異の発生源調査・封印

業務概要

人知を超えた未曾有の怪異や現象が発生した場合、その発生源を特定し、これ以上の被害拡大を防ぐための封印作業を行います。

迅速な現地調査と情報収集により、怪異の性質と行動パターンを分析。最も効果的な手段で、その存在をこの世から隔絶するか、力を無力化します。

実施手順

  1. 怪異発生に関する情報収集と初期分析
  2. 現場への専門チーム派遣と詳細調査
  3. 怪異の性質・発生源・影響範囲の特定
  4. 封印・無力化のための作戦立案
  5. 祓術・結界術を用いた封印作業の実施
  6. 周辺地域の安全確認と情報秘匿

こんな時にご相談ください

  • 地域で原因不明の集団幻覚や錯乱が発生している
  • 特定の場所で物理法則を無視した現象が繰り返し起きる
  • 未知の存在が目撃され、人々が恐怖を感じている
  • 従来の祓術では対応できないと判断された未解明の事象

相談・アドバイス（初回無料）

業務概要

「もしかして霊的なことかも…」「どこに相談すれば良いか分からない」といった漠然とした不安や、具体的な霊障の兆候を感じた際に、お気軽にご利用いただける初回無料の相談サービスです。

専門の構成員が丁寧にお話を伺い、状況を整理し、それが霊的な事象であるか、そうであるならばどのような対応が必要か、専門的な見地からアドバイスを提供します。

実施手順

  1. 専用フォームまたはメールにてご相談内容を送信
  2. 担当者よりご連絡、ヒアリングの日程調整
  3. オンラインまたは電話にて詳細なヒアリング（約30分）
  4. 専門的な見地からのアドバイスと今後の対応提案
  5. 必要であれば、正式な調査・祓術依頼への案内

こんな時にご相談ください

  • 特定の夢を繰り返し見るようになった
  • 動物が異常に怯える場所がある
  • 理由もなく気分が落ち込んだり、不眠が続く
  • 原因不明のものが壊れたり、物が移動したりする
  • 霊感があると感じるが、どうすればいいか分からない

まずはお気軽にご相談ください。専門の構成員があなたの不安に寄り添います。
